# Contributing to Idea Spark

Thank you for your interest in contributing! Your help makes Idea Spark better for everyone.

## How to Contribute

1. **Report Issues**
   - Found a bug or have a feature suggestion? Open an issue with clear steps or examples.

2. **Submit Pull Requests**
   - Fork the repository and create a branch for your changes.
   - Make sure your code follows existing style and is well-tested.
   - Submit a pull request describing your changes clearly.

3. **Coding Guidelines**
   - Use TypeScript and follow the existing React/Tailwind code style.
   - Keep commits small and focused.

4. **Stay Respectful**
   - Be respectful and considerate in discussions, reviews, and comments.
